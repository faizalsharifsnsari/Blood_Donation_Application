# Blood_Donation_Application
Blood Donation Application

Overview

The Blood Donation Application is designed to connect blood donors with those in urgent need of blood. This application simplifies the process of finding donors, organizing blood donation drives, and ensuring a reliable supply of life-saving blood.

Features

Donor Registration: Donors can register and update their details (blood type, location, availability).

Recipient Search: Patients or hospitals can search for donors based on blood type and location.

Request Management: Allows users to post and respond to urgent blood requests.

Notifications: Sends reminders to donors about upcoming donation opportunities.

Blood Drives: Displays upcoming blood donation events nearby.

Health Guidelines: Provides tips and eligibility criteria for blood donation.
